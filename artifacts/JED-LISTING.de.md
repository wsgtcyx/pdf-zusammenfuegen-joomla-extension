## Kurzbeschreibung

PDF Zusammenfuegen von pdfzus ist eine Joomla-Erweiterung fuer das lokale Zusammenfuehren mehrerer PDF-Dateien direkt im Browser. Die Dateien werden nicht an externe Server hochgeladen. Nutzer koennen PDFs sortieren, Seitenbereiche waehlen und das fertige Dokument ohne Wasserzeichen herunterladen.

## Vollbeschreibung

`PDF Zusammenfuegen von pdfzus` bringt einen datenschutzfreundlichen PDF-Merge-Workflow direkt in Joomla. Die Erweiterung richtet sich an Websites, die Besuchern oder internen Redakteuren ein schnelles Frontend-Werkzeug fuer Sammel-PDFs anbieten wollen, ohne dafuer einen Cloud-Upload oder einen externen Drittanbieter-Prozess zu erzwingen.

Die Erweiterung enthaelt eine eigenstaendige Joomla-Komponente fuer Menuepunkte und eine Content-Plugin-Einbettung ueber `{pdfzusmerge}`. Dadurch kann das Werkzeug sowohl als eigenstaendige Seite als auch innerhalb einer bestehenden Inhaltsseite genutzt werden.

### Kernfunktionen

- Mehrere PDFs lokal im Browser auswaehlen
- Reihenfolge der Dateien per Drag-and-Drop aendern
- Seitenbereiche pro Datei optional eingrenzen
- Zusammengefuehrtes PDF direkt herunterladen
- Kein Wasserzeichen im Ergebnis
- Keine Pflichtregistrierung
- Keine Telemetrie und keine PDF-Uebertragung an Fremdserver

### Datenschutz und DSGVO

Die eigentliche Merge-Verarbeitung findet lokal im Browser statt. Die Erweiterung ist so konzipiert, dass PDF-Dateien den Joomla-Server nicht verlassen muessen. Damit eignet sie sich besonders fuer datensensible Einsatzbereiche wie Bewerbungsunterlagen, interne Sammeldokumente, Rechnungen oder Versandpakete fuer Kunden.

### Typische Einsatzfaelle

- Bewerbungsunterlagen als eine Datei zusammenstellen
- Rechnungen und Belege fuer Buchhaltung sammeln
- Projektanlagen oder Vertragsdokumente in eine Versanddatei ueberfuehren
- Interne Dokumentensets auf einer Joomla-Seite bereitstellen

### Was die Erweiterung bewusst nicht macht

- kein Upload an pdfzus.de
- kein erzwungener Entwickler-Backlink im Frontend
- keine Wasserzeichen
- keine Kontopflicht
